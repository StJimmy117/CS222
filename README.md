# CS222

A small Python utility for querying the most recent revision history of a Wikipedia article.

## Features
- Fetches up to 30 most recent changes for a given Wikipedia article
- Detects redirects and prints the redirected article name
- Handles missing pages and network errors with distinct exit codes

## Usage
Run the script with the article name as command-line arguments:

```bash
./wikipedia_changes.py Ball State University
```
